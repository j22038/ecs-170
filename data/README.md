# Data Information

This project uses two sets of images:
- `data/raw/` — original images
- `data/processed/` — resized 128x128 grayscale images

Only a small sample is included here.  
The full dataset (~2GB) can be downloaded from: https://drive.google.com/drive/folders/1ARG2XZil9Ft9r4p6ZxtgPLVP-8kdWs9l?usp=drive_link
